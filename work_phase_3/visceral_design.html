Visceral Design

Visceral design is the first level of design, focusing on the immediate emotional response a user has to a product. It encompasses the initial impressions and feelings evoked by the product's appearance, feel, and sound..

Colour Scheme

I used brown as the base colour for the my design. The reason for this is that brown is a colour that is associated with warmth, comfory. As the website aims to be a inclusive and welcoming space for all users, providing them with a sense of comfort and security can encourge them to revisit the site.
Moreover, the colour brown also associated withe the nature and healthy, which also can relate to the healthy lifestyle the website want to promote.
As result, choosing brown is as the main colour can associated with the value of the website.


The colour scheme is as follows:

(#6B331E) - Warm brown

Secondary colour
(#20282E) - Light brown

Accent colour
(#331107) - Rich earthy brown

Text colour
(#170300) - Deep brown

Background colour
(#F2F1EF) - Soft off-white

Typography

The font I used for the website is "Nunito", sans-serif. As the website aims to be a welcoming and inclusive space for all users, I chose this font because it is a sans-serif font that is easy to read and has a friendly and approachable feel.
The font is also modern and clean, which aligns with the website's goal of promoting a healthy lifestyle.
The font size follow the golden ratio, which is a design principle that is often used in design to create a sense of harmony and balance. I set the font size in screen size to be 1rem and mobile to be 0.75rem, and times 1.618 for the next size. Up to 4.2 rem for screen size and 3.5rem for mobile size.
This time I decided to change the font weight only when needed to maintain the harmony of the design.
The font weight is set to 400 for normal text and 700 for headings.
